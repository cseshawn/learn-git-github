# Git Branching & Collaboration

## 1. View Branches

```bash
git branch
```

List all local branches.

---

## 2. Create a Branch

```bash
git branch feature-login
```

Create a new branch.

---

## 3. Switch Branch

```bash
git switch feature-login
```

or

```bash
git checkout feature-login
```

---

## 4. Create & Switch Together

```bash
git switch -c feature-login
```

or

```bash
git checkout -b feature-login
```

---

## 5. Rename Branch

Current branch:

```bash
git branch -m new-name
```

Specific branch:

```bash
git branch -m old-name new-name
```

---

## 6. Delete Branch

```bash
git branch -d feature-login
```

Force delete:

```bash
git branch -D feature-login
```

---

## 7. Compare Branches

```bash
git diff main feature-login
```

See differences between branches.

---

## 8. Merge Branch

Switch to target branch first:

```bash
git switch main
git merge feature-login
```

Merge `feature-login` into `main`.

---

## 9. Merge Conflict

Occurs when Git cannot automatically combine changes.

Example:

```bash
git merge feature-login
```

Resolve conflicts manually, then:

```bash
git add .
git commit
```

---

## 10. Remote Branches

View remote branches:

```bash
git branch -r
```

View all branches:

```bash
git branch -a
```

---

## 11. Push New Branch

```bash
git push -u origin feature-login
```

`-u` sets the upstream branch.

---

## 12. Upstream Branch

Check tracking info:

```bash
git branch -vv
```

Set upstream manually:

```bash
git branch --set-upstream-to=origin/main
```

---

## 13. Pull Latest Changes

```bash
git pull
```

Fetch and merge remote updates.

---

## 14. Pull Request (PR)

Typical workflow:

```text
Create Branch
    ↓
Make Changes
    ↓
Commit
    ↓
Push Branch
    ↓
Open Pull Request
    ↓
Code Review
    ↓
Merge
```

A Pull Request allows others to review changes before merging.

---

## Branch Workflow

```text
main
  │
  └── feature-login
          │
          ├── commit
          ├── commit
          │
          └── merge → main
```

```bash
git branch

git switch -c new-branch
git merge branch-name
git diff main feature
git branch -d branch-name

```
