# Git Branch Workflow

## 1. View Branches

```bash
git branch      # Local branches
git branch -r   # Remote branches
git branch -a   # All branches
```

---

## 2. Create a New Branch

```bash
git branch feature-login
```

Create only.

Create and switch:

```bash
git switch -c feature-login
```

or

```bash
git checkout -b feature-login
```

---

## 3. Switch Between Branches

```bash
git switch main
git switch feature-login
```

or

```bash
git checkout main
```

---

## 4. Rename a Branch

Current branch:

```bash
git branch -m new-name
```

Specific branch:

```bash
git branch -m old-name new-name
```

---

## 5. Work and Commit

```bash
git add .
git commit -m "Add login feature"
```

---

## 6. Compare Changes

Compare branches:

```bash
git diff main feature-login
```

Compare working directory:

```bash
git diff
```

---

## 7. Push Branch to GitHub

```bash
git push -u origin feature-login
```

### Upstream

The `-u` flag connects the local branch with the remote branch.

Check tracking branches:

```bash
git branch -vv
```

---

## 8. Pull Request (GitHub)

Workflow:

```text
Create Branch
    ↓
Make Changes
    ↓
Commit
    ↓
Push Branch
    ↓
Open Pull Request
    ↓
Code Review
    ↓
Merge
```

A Pull Request (PR) is used to review changes before merging them into the main branch.

---

## 9. Pull Latest Changes

```bash
git pull
```

Fetches and merges updates from the remote repository.

---

## 10. Merge a Branch

Switch to the target branch first:

```bash
git switch main
git merge feature-login
```

Result:

```text
main + feature-login → main
```

---

## 11. Resolve Merge Conflicts

Conflict example:

```bash
git merge feature-login
```

Git may show:

```text
CONFLICT (content): Merge conflict
```

Steps:

```bash
# Edit conflicted files
git add .
git commit
```

---

## 12. Delete Branch

After successful merge:

```bash
git branch -d feature-login
```

Force delete:

```bash
git branch -D feature-login
```

Delete remote branch:

```bash
git push origin --delete feature-login
```

---

# Complete Workflow

```text
View Branches
      ↓
Create Branch
      ↓
Switch Branch
      ↓
Code Changes
      ↓
Commit
      ↓
Diff (Optional)
      ↓
Push Branch
      ↓
Pull Request
      ↓
Merge
      ↓
Resolve Conflicts (if any)
      ↓
Delete Branch
```
