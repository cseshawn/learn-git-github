## ⚡ Git Command Cheat-Sheet

| Command                           | Description                                      |
| :-------------------------------- | :----------------------------------------------- |
| `git init`                        | Initialize a new Git repository                  |
| `git add <file>`                  | Stage a file for the next commit                 |
| `git add .`                       | Stage all changes in the current directory       |
| `git commit -m "message"`         | Commit staged changes with a message             |
| `git status`                      | Show the current state of the working directory  |
| `git log --oneline`               | View a condensed history of commits              |
| `git log --graph --oneline --all` | View a visual representation of branch history   |
| `git branch <branch-name>`        | Create a new branch                              |
| `git switch <branch-name>`        | Switch to a different branch                     |
| `git merge <branch-name>`         | Merge another branch into the current one        |
| `git remote add origin <url>`     | Link your local repo to a remote on GitHub       |
| `git push -u origin main`         | Push local commits to the remote `main` branch   |
| `git pull`                        | Fetch and integrate changes from the remote repo |
| `git clone <url>`                 | Copy a remote repository to your local machine   |
