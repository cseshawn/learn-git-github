# Git vs GitHub

| Git                          | GitHub                              |
| ---------------------------- | ----------------------------------- |
| Version Control System (VCS) | Cloud platform for Git repositories |
| Runs on your computer        | Runs online                         |
| Tracks code changes          | Stores and shares repositories      |
| Works offline                | Requires internet for sync          |
| Used for version management  | Used for collaboration and hosting  |

## Common Git Commands

```bash
git init          # Initialize repository
git status        # Check repository state
git add .         # Stage changes
git commit -m ""  # Save changes locally
git log           # View history
git branch        # Manage branches
git merge         # Merge branches
```

## Common GitHub Commands

```bash
git clone <repo-url>   # Download repository
git push origin main   # Upload changes
git pull origin main   # Get latest changes
```

## Basic Workflow

```bash
git init
git add .
git commit -m "First commit"

git remote add origin <repo-url>
git push -u origin main
```

## Summary

- **Git** = Tracks and manages code versions locally.
- **GitHub** = Hosts Git repositories online for sharing and collaboration.
- **Git + GitHub** = Version control + team collaboration.
